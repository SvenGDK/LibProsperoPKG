# HomebrewTest

A minimal free homebrew module for a PS5 console, plus the debug fPKG built from it.

The module renders three centered text lines through the platform video-out path with a CPU-written framebuffer.

## Files

| File | Purpose |
|---|---|
| `eboot.bin` | Compiled 64-bit ELF module. |
| `sce_sys/param.json` | Package metadata. |
| `sce_sys/icon0.png` | Package icon. |
| `UP9000-PPSA99099_00-PROSPERO00000000-A0100-V0100.pkg` | The built debug fPKG. |
